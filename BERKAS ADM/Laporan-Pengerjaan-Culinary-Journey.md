# 1\. PENDAHULUAN

## 1.1 Gambaran Umum Proyek

Culinary Journey: Menjelajahi Rasa Nusantara adalah sebuah web story interaktif yang menghadirkan pengalaman eksplorasi kuliner dari tiga daerah ikonik Indonesia: Yogyakarta, Sumatera Selatan, dan Nusa Tenggara Barat.

Format output yang dipilih adalah web story / visual essay scrollytelling yang menggabungkan narasi, visualisasi data, dan desain interaktif untuk menciptakan pengalaman yang imersif dan berbasis data nyata dari dataset Kaggle.

## 1.2 Tujuan Proyek

Proyek ini memiliki tujuan sebagai berikut:

* Menghadirkan platform digital yang mengintegrasikan data kuliner tiga daerah dalam format interaktif dan menarik

* Mendokumentasikan warisan kuliner Indonesia secara berbasis data dengan visualisasi yang komprehensif

* Mempromosikan potensi kuliner dan pariwisata ketiga daerah kepada audiens yang lebih luas

* Mendemonstrasikan implementasi praktis dari storytelling berbasis data menggunakan HTML5, CSS3, JavaScript, dan library visualisasi

# 2\. KERANGKA KERJA (REVIEW)

## 2.1 Ringkasan Requirements

Berdasarkan dokumen kerangka kerja yang telah disusun, berikut adalah ringkasan functional dan non-functional requirements:

Functional Requirements:

* FR-01: Halaman Landing dengan hero section, animasi visual, dan CTA

* FR-02: Eksplorasi Daerah (3 halaman) dengan card hidangan, filter, flavor bar

* FR-03: Halaman Detail Hidangan dengan radar chart 5 dimensi rasa dan bar chart nutrisi

* FR-04: Halaman Data & Insight dengan visualisasi perbandingan antar daerah

* FR-05: Navigasi konsisten, scrollytelling, dan responsivitas penuh

Non-Functional Requirements:

* Performa: Loading \< 3 detik, animasi ≥ 60 fps, bundle \< 500 KB gzipped

* Responsivitas: Mobile (\< 768px), tablet (768–1024px), desktop (\> 1024px)

* Aksesibilitas: Contrast ratio ≥ 4.5:1, alt text untuk gambar, navigasi keyboard

* Browser compatibility: Chrome, Firefox, Safari, Edge (2 versi terakhir)

## 2.2 Template Referensi Terpilih

Template terpilih untuk referensi desain adalah:

🥇 Pilihan Utama: Bestro Restaurant Bootstrap Template

Alasan: Layout multi-halaman, card grid untuk hidangan, hero slider, struktur navigation sesuai dengan kebutuhan web story.
